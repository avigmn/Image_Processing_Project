# AI Assistance Log

Throughout this project, I used two AI tools - **Google Gemini** and **Claude (Anthropic)** - for a variety of tasks. Here's a summary of how they helped:

---

## 1. Documentation & README
AI helped write the README and project documentation - structure, model descriptions, run instructions, and the improvements table. AI also helped write this AI Assistance Log. I reviewed everything to make sure it matched what the code actually does.

## 2. Code Implementation
I consulted AI when figuring out how to implement parts of the algorithm - things like 3D DCT feature extraction, Mutual Information feature selection, Naive Bayes, SVM, Logistic Regression, and the ensemble majority vote. AI suggested approaches, I tested them and adjusted based on results.

## 3. Debugging
AI helped diagnose several bugs along the way - for example, a sign inconsistency in the DCT+LogReg model (using signed vs. absolute values), a misconfigured confidence threshold, and cross-contamination between action classes. I confirmed each fix by running the pipeline and checking the output videos.

## 4. Code Review
AI reviewed the code for missing steps, inconsistencies between training and inference, and alignment with the original paper. I cross-referenced with the paper and validated results manually.

## 5. GitHub
I used AI to learn how to work with GitHub - cloning, committing, pushing, setting up `.gitignore`, and organizing the repository.
